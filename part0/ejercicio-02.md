# CSS
---

Css no es un lenguaje de marcado ni de programacion, es un lenguaje de estilos. 

* Las reglas al declarar una sentencia van con corchetes. 
* Estas pueden ir concatenadas de varios elementos, es decir p, h1, h2 etc. Y de ese modo se les puede asignar una propiedad de manera general.

*Existen tambien selectores como: 
    * < p > Todos los elementos del HTML del tipo
    * < p  id = "mi_id"> El elemento con el id especificado
    * < p class = "mi_clase"> Clase especificada
    * < img src="imagen.png"> Atributo especificado
    * < a:hover> Se selecciona solo cuando el mouse está en él

Existen muchas formas de estilizar y dar vida a una pagina web, esto resume lo basico para entender css. Existen propiedades para cambiar el tamaño de letra, dar color a la misma, cambiar el fondo de pantalla etc.
