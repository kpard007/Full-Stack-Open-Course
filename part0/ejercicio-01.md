# HTML
--- 

En este ejercicio nos muestran que es HTML (Lenguaje de marcado)
y como se pueden crear titulos, parrafos o listas.

* Para crear una etiqueta esta siempre tiene un inicio '<' y un cierre '/>'.
* Existen etiquetas para parrafos '<p>' y tambien para titulos <h1>, <h2>, <h3>, etc.
* Tambien existen para cargar imagenes '<img>'
    * A las etiquetas se les pueden añadir atributos con los cuales se le pueden dar estilos.
* A la etiqueta de imagen se le puede añadir un atributo de 'alt' para añadir un mensaje.
    * Este mensaje nos puede ayudar en caso de error (como que la imagen no se muestre).
    * Puede ayudar a personas que tengan una discapacidad y les sea mas facil entender.
* La etiqueda '<a>' funciona para hacer referencia a links por ejemplo: 
    * <a href ="pagina_receta.html">Pagina_receta</a>
